# Playdate Pinball Prototype (Portrait)

Second Playdate prototype focused on a **long-ways / portrait-feel** playfield.

## Features (v0)
- Tall playfield concept (within native 400x240 landscape render)
- Ball physics (gravity, wall collisions)
- Left/right flippers
- One bumper + score
- Drain + auto reset

## Controls
- Left: D-pad Left
- Right: D-pad Right
- Launch: A
- Reset ball: B

## Build

```bash
./build.sh
```
